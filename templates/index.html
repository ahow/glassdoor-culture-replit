About This Dashboard

This dashboard analyzes employee reviews from Glassdoor to extract and visualize organizational culture insights for financial services companies. The system uses two established culture analysis frameworks to provide a comprehensive view of workplace culture.

Hofstede Framework

6 bipolar dimensions measuring organizational culture on a -1 to +1 scale. This framework examines aspects like power distance, individualism vs. collectivism, and uncertainty avoidance.

Learn more about Hofstede's model

MIT Big 9 Framework

9 unipolar dimensions measuring culture attributes on a 0-10 scale. Covers agility, collaboration, customer orientation, diversity, execution, innovation, integrity, performance, and respect.

Learn more about MIT's Culture 500

How It Works

  1. Data Collection: Reviews are extracted from Glassdoor using their public data
  2. Culture Scoring: Each review is analyzed using keyword-based scoring algorithms for both frameworks
  3. Aggregation: Individual scores are aggregated to company-level metrics with confidence indicators
  4. Visualization: Results are presented through interactive charts for comparison and trend analysis
